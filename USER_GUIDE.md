## ğŸ”§ Troubleshooting

### Camera not working

**Reason 1: Permissions not granted**
- Check browser settings
- Grant camera permission
- Reload the page

**Reason 2: Camera is busy**
- Close other apps using camera
- Close other browser tabs with camera
- Restart device

**Reason 3: Browser not supported**
- Use Chrome, Safari, or Firefox
- Update browser to latest version

### Objects not adding

1. Reload the page
2. Clear browser cache
3. Check internet connection (for loading SVGs)

### Photos not saving

1. Check browser storage space
2. Don't use incognito/private mode
3. Allow data storage for the site

### App running slowly

1. Close other browser tabs
2. Close background apps
3. Restart device
4. Reduce number of objects on screen
